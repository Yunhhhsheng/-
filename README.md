## 项目内容

本项目旨在复现发表于 *Catena* 2024 年的论文 **“Precipitation rather than temperature primarily drives global termite effects on litter decomposition”** 中的核心图表（图3、图4）。该研究通过荟萃分析评估了全球尺度下白蚁对凋落物分解的影响，并识别了关键气候驱动因子（如年均降水量 MAP、年均温度 MAT 等）。我们将基于原文提供的数据图表数字化和统计方法，尝试复现其主要结果。

### 复现目标

- **图3**：展示不同生态系统、基质类型、排除方法和分解环境下白蚁对凋落物质量损失的响应比（lnRR）及其 95% 置信区间。
- **图4**：展示在不同生态系统、基质类型、研究方法等分组下，气候与地理因子（如 MAT、MAP、纬度、海拔等）对 lnRR 的影响方向与显著性。

### 数据来源

- 原始数据来源：由于作者未提供原始数据，数据从图表中数字化提取，部分可从补充材料中获取

### 结果解读
#### 图3解读

图3展示了白蚁对不同条件下凋落物分解的影响。结果显示，白蚁的存在使全球凋落物质量损失率平均提高 **41.2%**（图3a）。在不同生态系统中，白蚁对森林和草地凋落物分解的促进作用最显著（分别使质量损失率提高44.8%和30.4%），而在农田和沙漠中效果不显著。从基质类型看，木质凋落物（树枝、树干）、草类和粪便在缺乏白蚁时分解速率下降超过42%，而根、叶和秸秆的分解受白蚁影响不显著（图3b）。此外，白蚁在土壤表面的促进作用（+43.4%）远强于埋入土壤（+22.1%，但不显著），且对于干次湿润地区的效果最明显。

#### 图4解读

图4分析了不同环境因子对白蚁效应（lnRR）的调节作用。在森林生态系统中，年均温（MAT）、年均降水量（MAP）和经度对白蚁促进分解呈正相关，而纬度、海拔和蒸发量呈负相关；在草地中，MAT和海拔为正相关，蒸发量为负相关（图4a）。对于木质凋落物，MAT、MAP和经度促进分解，而海拔和纬度抑制分解；叶片凋落物则受MAT、海拔和试验时长的正向影响（图4b）。在不同排除方法中，网袋法下MAT、MAP和经度为正相关，而杀虫剂法下仅纬度为正相关、蒸发量为负相关（图4c）。总体而言，MAP是影响白蚁分解作用的最关键驱动因子。
